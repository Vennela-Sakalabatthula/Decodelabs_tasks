# Project 1: User Persona & Empathy Map for a Fitness App

## Overview

This project focuses on understanding the needs, goals, behaviors, and challenges of busy professionals through UX research techniques. A User Persona and Empathy Map were created to gain insights into user pain points before designing a fitness application.

## Problem Statement

Busy professionals often struggle to maintain fitness due to hectic schedules, stress, and lack of time. Understanding their needs is essential for creating an effective and user-friendly fitness solution.

## Objective

To analyze user behavior and identify opportunities for improving fitness habits through a user-centered approach.

## Deliverables

* User Persona
* Empathy Map
* Problem Statement
* Target Audience Analysis
* App Goal Definition

## Key Learnings

* User Research
* Persona Creation
* Empathy Mapping
* User-Centered Design
* Problem Identification
